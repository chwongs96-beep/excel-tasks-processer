## 1. 這套工具能做什麼

本工具協助您完成證券會計相關的 **Excel 資料處理與報表產出**，主要能力包括：

| 功能 | 說明 |
|------|------|
| 匯入 Excel | 單檔、多檔、整個資料夾 |
| 檔名關鍵字篩選 | 資料夾內只抓「檔名符合規則」的檔案 |
| 欄位映射 | 將各種券商／系統欄名對到標準會計欄位 |
| 驗證與預覽 | 產報前檢查資料與轉換結果 |
| 產生日／週／月報 | 依範本輸出正式報表 |
| 多檔合併 | 多個 Excel 合併成一個檔案 |
| 雙邊對帳 | 比對兩套來源的鍵值與金額差異 |
| 批次日報 | 依日期區間一次產多份日報 |
| 調整分錄 | 另載調整表，產報時一併併入 |

**不適合**的狀況：若您需要「原封不動搬移 Excel 分頁（含公式、圖表）」，請使用 Excel 內建的「移動或複製工作表」，見 [第 7 節](#7-多檔合併為單一-excel)。

## 2. 安裝與啟動

### 2.1 環境需求

- **Windows 10/11**（建議）
- **Python 3.11 或以上**

### 2.2 第一次安裝

在專案目錄開啟 PowerShell：

```powershell
cd report_desktop_app
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

若提示缺少父目錄套件，請回到上一層再執行：

```powershell
cd ..
pip install -r requirements.txt
```

### 2.3 啟動程式

```powershell
cd report_desktop_app
.\.venv\Scripts\activate
python main.py
```

或於 repo 根目錄：

```powershell
.\scripts\run_desktop.ps1
```

### 2.4 測試用範例檔

`report_desktop_app/tests/fixtures/sample_ledger.xlsx` — 可用來熟悉映射與產報流程。

## 4. 標準流程：產生日／週／月報

這是日常最常用的路徑，請依序執行。

### 步驟 ① 匯入 Excel

1. 左側 **＋ 新增檔案**，選擇一個或多個 `.xlsx` / `.xls`。
2. 或按 **資料夾**，用關鍵字篩選後匯入（見 [第 5 節](#5-智慧資料夾匯入檔名關鍵字)）。
3. 匯入時會顯示 **處理進度** 視窗，列出每一檔的讀取步驟。
4. 確認左側清單出現檔名與列數。

### 步驟 ② 設定範圍（建議）

若 Excel 不是從 A1 開始，或只要部分列：

1. 在清單中**點選檔案**（或連按兩下）。
2. 按 **設定範圍**（或工具列上的範圍按鈕）。
3. 指定：**工作表**、**標題列**、起迄列，或直接輸入如 `B2:H500`。
4. 按 **預覽範圍** 確認欄位正確 → **確定**。

可載入／儲存 **範圍 preset**（例如固定某券商的版面）。

### 步驟 ③ 欄位映射

1. 選取要設定的檔案，按 **欄位映射**。
2. 為每個「標準欄位」選擇對應的 Excel 欄位；留空表示略過。
3. 可 **載入映射 preset**（`config/mapping_presets/`）或 **儲存** 目前設定。
4. 按 **確定**。

> 映射只需設定一次；多檔若欄名相同，通常沿用同一套 preset。

### 步驟 ④ 驗證資料

1. 在 **② 報表設定** 選擇報表類型與**期間**（日／週／月）。
2. 按底部 **驗證資料**。
3. 若有錯誤，日誌與對話框會說明；請修正映射或範圍後再試。

### 步驟 ⑤ 預覽轉換

1. 按 **預覽轉換**。
2. 到中央 **「轉換後」** 分頁查看標準化後的資料列數與欄位。
3. 確認無明顯異常（空白過多、日期錯誤等）。

### 步驟 ⑥ 產生報表

1. 確認 **範本** 與 **輸出** 路徑正確。
2. 按 **產生報表**。
3. 完成後可按 **輸出夾** 開啟資料夾，或依對話框路徑開啟檔案。

```text
匯入 → 範圍 → 映射 → 驗證 → 預覽 → 產報
```

## 5. 智慧資料夾匯入（檔名關鍵字）

適用情境：每天把多個 Excel 丟進同一資料夾，但只想自動處理「檔名含特定關鍵字」的檔案。

### 操作步驟

1. 左側按 **資料夾**。
2. 選擇資料夾路徑。
3. 設定規則：
   - **包含關鍵字**：逗號分隔，例如 `成交, TDCC`（留空＝全部 Excel）
   - **排除關鍵字**：例如 `draft, 備份, temp`
   - **不分大小寫**：建議勾選
4. 按 **預覽符合檔案** — 查看 ✓ 將匯入／－ 將略過。
5. （可選）**載入／儲存關鍵字 preset**。
6. （可選）**匯入後套用範圍** — 選已儲存的範圍 preset，匯入後自動套到每個檔案。
7. 按 **匯入符合的檔案**。
8. （可選）勾選 **啟用監看** — 每 60 秒自動依相同規則匯入新檔。

### 範例

| 目的 | 包含 | 排除 |
|------|------|------|
| 日結成交明細 | `成交` | `draft, 備份` |
| 特定券商 | `富邦, 元大` | `temp` |

### Preset 儲存位置

`config/file_filter_presets/`（JSON 檔，可備份與分享給同事）。

## 6. 設定資料範圍與清除內容

### 6.1 設定範圍

- 入口：**設定範圍**、清單**連按兩下**檔案、合併精靈步驟 2。
- **Excel 範圍**（如 `A2:Z9999`）優先於手動列號。
- **範圍 preset** 可儲存在 `config/range_presets/`。

### 6.2 清除範圍內容

若要在寫入前清掉某區塊的舊資料（僅清除**儲存格的值**，直接寫回檔案）：

**方式 A**：左側選檔 → **清範圍** → 指定範圍 → 確認。

**方式 B**：在「設定範圍」對話框內按 **清除此範圍內容…**。

> ⚠️ **無法復原**，執行前請確認範圍與檔案無誤，建議先備份原檔。

## 7. 多檔合併為單一 Excel

### 7.1 先選做法

| 需求 | 建議 |
|------|------|
| 多檔明細接成**一張總表** | 合併 → **合併到單一工作表**（會加 `_source_file` 欄） |
| 每檔保留**獨立分頁** | 合併 → **每個來源一個工作表** |
| 保留公式／完整格式搬分頁 | 用 **Excel 內建**「移動或複製工作表」 |

### 7.2 本工具合併步驟

1. **匯入**所有來源檔（或從資料夾關鍵字匯入）。
2. 每檔 **設定範圍**。
3. 按 **合併**：
   - **智慧合併建議**：回答目標與條件，系統建議單表或分頁模式。
   - **合併精靈**三步：確認檔案 → 各檔範圍 → 輸出名稱與模式。
4. 處理進度視窗會顯示：檢查輸出 → 逐檔讀取 → 寫入合併檔。
5. 可勾選：
   - **合併完成後自動匯入**
   - **開啟欄位映射**（接續產報）

詳細圖文亦可在程式內：**說明 → 合併與工作表流程**。

### 7.3 Excel 內建搬移分頁

1. 同時開啟來源與目標活頁簿。
2. 來源檔工作表標籤 **右鍵** → **移動或複製…**。
3. 選目標活頁簿與位置；要保留來源請勾 **建立副本**。

## 8. 資料對帳

比對**兩個**已匯入的 Excel（例如券商明細 vs 內部系統）。

### 操作步驟

1. 先匯入**左檔（基準）**與**右檔**，並各自設定範圍。
2. 左側按 **對帳**。
3. 在對話框：
   - **① 說明與原理**：了解僅左／僅右／金額不符的意義。
   - **② 對帳設定**：選左檔、右檔、對帳鍵欄位、金額欄、容差。
   - 可使用 **建議鍵** 自動勾選常見欄位組合。
4. 按 **執行對帳**；可勾選匯出 Excel 報告。
5. 到主畫面 **「對帳差異」** 分頁預覽結果。

### 對帳結果怎麼讀

| 類型 | 意義 | 常見原因 |
|------|------|----------|
| 僅左邊 | 左有、右無 | 右檔漏範圍、漏登 |
| 僅右邊 | 右有、左無 | 重複登錄、左檔缺漏 |
| 金額不符 | 鍵相同、金額差超過容差 | 手續費、匯率、映射錯欄 |

程式內說明：**說明 → 對帳功能說明**。

## 9. 其他功能

### 9.1 調整分錄

1. 左側 **調整**，選擇調整分錄 Excel。
2. 設定範圍（與一般檔案相同）。
3. 產報時會與來源明細合併，並標記 `_entry_type=adjustment`。

### 9.2 批次產生日報

1. 完成匯入與映射。
2. 底部 **批次日報**。
3. 選日期區間；可勾 **僅工作日**。
4. 系統依每個日期各產一份日報至輸出資料夾。

### 9.3 移除與全清

- **移除**：刪除清單中選取的單一檔案。
- **全清**：移除工作階段中所有已匯入檔案（需確認）。

### 9.4 資料夾監看

在「資料夾匯入」勾選 **啟用監看** 後，程式每 **60 秒** 掃描資料夾，依**相同關鍵字規則**自動匯入新出現的 Excel。

## 13. 常見問題（FAQ）

**Q：匯入後預覽是空的？**  
A：檢查是否設錯工作表或範圍；按「預覽範圍」確認標題列是否正確。

**Q：驗證失敗「缺少必要欄位」？**  
A：開啟欄位映射，補齊必填欄位（見 `canonical_schema.yaml` 中的 required 欄位）。

**Q：合併後列數不對？**  
A：逐檔檢查標題列、結束列是否含大量空白列；必要時縮小 Excel 範圍。

**Q：對帳鍵太多重複？**  
A：增加對帳鍵欄位（如流水號）；或先篩選單一日期再對帳。

**Q：資料夾匯入沒有檔案？**  
A：確認關鍵字是否太嚴格；按「預覽符合檔案」看被排除的檔名。

**Q：程式顯示「正在處理上一項操作」？**  
A：等待目前工作完成（處理進度視窗關閉）；勿連續快速點擊產報。

**Q：可以同時開兩個程式嗎？**  
A：不建議對同一輸出檔或同一 Excel 來源同時寫入，可能造成檔案鎖定或覆蓋。

## 14. 建議日常作業順序

### 情境 A：每日日報（固定格式）

```text
資料夾（關鍵字）→ 範圍 preset 自動套用 → 映射 preset
→ 驗證 → 預覽 → 產報
```

### 情境 B：月底多檔合併再產報

```text
匯入多檔 → 各檔範圍 → 合併（單表）→ 映射 → 驗證 → 預覽 → 產報
```

### 情境 C：與券商檔對帳

```text
匯入兩檔 → 範圍 → 對帳 → 檢視「對帳差異」→ 匯出覆核表
```
